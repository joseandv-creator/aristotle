// WHAT IS THIS
// QUE ES ESTO

A protocol for discovering what is true regardless of who observes it. Not a collection of opinions. Not a forum. A filter.

Anyone can propose a candidate universal truth. The system decomposes it, scores its fidelity to reality, tests it across multiple domains, and delivers a verdict. What survives gets integrated. What doesn't shows you exactly why it failed.

// the goal is not to be right. it's to see more clearly.

Un protocolo para descubrir lo que es verdad sin importar quien lo observe. No es una coleccion de opiniones. No es un foro. Es un filtro.

Cualquiera puede proponer una verdad universal candidata. El sistema la descompone, puntua su fidelidad a la realidad, la prueba en multiples dominios, y entrega un veredicto. Lo que sobrevive se integra. Lo que no, te muestra exactamente por que fallo.

// el objetivo no es tener razon. es ver con mas claridad.

// WHAT IS IT FOR
// PARA QUE SIRVE

Truths are not the end. They are the tool.

The end is that each subject reaches their object across any time horizon. The long term dominates the short — that is universal truth. But which long term is yours to decide — that is will. Where those two variables cross, real navigation is born: structure tells you what to prioritize, you decide where to go.

Las verdades no son el fin. Son la herramienta.

El fin es que cada sujeto alcance su objeto en cualquier horizonte temporal. El largo plazo domina al corto — eso es verdad universal. Pero cual largo plazo es tu decision — eso es voluntad. Donde esas dos variables se cruzan, nace la navegacion real: la estructura te dice que priorizar, tu decides hacia donde ir.

// HOW IT WORKS
// COMO FUNCIONA
// the protocol operates in 6 steps 1. DECOMPOSITION your candidate is separated into: subject, object, relation, value sought 2. SCORING each component is scored from 0.0 to 1.0 the total score is the product — if anything is zero, everything collapses 3. UNIVERSALITY tested across 4+ unrelated domains if it doesn't survive in biology, economics, physics AND psychology it's not universal — it's local 4. INHERITANCE where does this idea come from? did you derive it or inherit it? a belief repeated a thousand times is not a truth 5. REDUNDANCY does this truth already exist in another form? if so, does your formulation add something new? 6. VERDICT COLLAPSE = rejected PARTIAL = dangerous, looks true but fails under pressure PROVISIONAL = viable, needs more territory NAVIGABLE = solid, ready to use CLOSED = its negation destroys itself
// el protocolo opera en 6 pasos 1. DESCOMPOSICION tu candidata se separa en: sujeto, objeto, relacion, valor buscado 2. PUNTUACION cada componente se puntua de 0.0 a 1.0 la puntuacion total es el producto — si algo es cero, todo colapsa 3. UNIVERSALIDAD se prueba en 4+ dominios no relacionados si no sobrevive en biologia, economia, fisica Y psicologia no es universal — es local 4. HERENCIA de donde viene esta idea? la derivaste o la heredaste? una creencia repetida mil veces no es una verdad 5. REDUNDANCIA esta verdad ya existe en otra forma? si es asi, tu formulacion agrega algo nuevo? 6. VEREDICTO COLAPSO = rechazada PARCIAL = peligrosa, parece verdad pero falla bajo presion PROVISIONAL = viable, necesita mas territorio NAVEGABLE = solida, lista para usar CERRADA = su negacion se destruye a si misma
